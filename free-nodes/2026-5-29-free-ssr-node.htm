  • Mac用户：下载.dmg镜像文件。
  • Android和iOS用户：可以直接在应用商店中搜索“筋斗云”进行下载安装。

第二步：安装筋斗云

不同操作系统的安装方法略有不同，但都非常简单。

对于Windows用户： 1. 双击下载的安装包（.exe文件）。 2. 系统可能会弹出安全提示，点击“是”继续。 3. 按照安装向导的提示，选择安装路径，点击“下一步”直到完成安装。 4. 安装完成后，桌面会出现筋斗云的快捷图标。

对于Mac用户： 1. 打开下载的.dmg文件。 2. 将筋斗云图标拖入“应用程序”文件夹。 3. 首次打开时，系统可能会提示“此应用来自不明开发者”，请前往“系统偏好设置”中的“安全性与隐私”允许运行。 4. 之后，你可以在启动台中找到筋斗云并打开。

对于Android和iOS用户： 1. 打开应用商店（如Google Play或App Store）。 2. 搜索“筋斗云”，找到官方应用。 3. 点击“安装”或“获取”，等待下载完成。 4. 安装完成后，应用图标会出现在手机桌面上。

第三步：注册账号

安装完成后，打开筋斗云客户端。首次使用会提示你注册账号。点击“注册”按钮，根据提示填写相关信息，如邮箱地址、用户名和密码。建议使用真实有效的邮箱，以便接收重要通知或找回密码。填写完成后，点击“注册”即可。

第四步：登录账号

注册成功后，返回登录界面。输入你刚刚注册的邮箱和密码，点击“登录”。如果你已经拥有账号，直接登录即可。

第五步：连接VPN

登录成功后，你会看到客户端界面显示多个服务器节点。这些节点分布在全球各地，如美国、日本、新加坡、香港等。选择离你地理位置最近的节点，通常可以获得最快的连接速度。点击节点旁边的“连接”按钮，等待几秒钟，状态变为“已连接”即可开始使用。

第四章：筋斗云的配置教程——优化你的连接体验

虽然筋斗云开箱即用，但通过一些配置优化，你可以获得更好的连接体验。以下是一些实用的配置建议。

配置最佳节点

节点的选择直接影响连接速度。一般来说，选择离你物理位置最近的节点可以获得最低的延迟。例如，如果你在中国大陆，可以选择香港、新加坡或日本的节点；如果你在北美，可以选择美国西海岸的节点。筋斗云客户端通常会显示每个节点的延迟时间，你可以根据这个数据做出选择。

此外，如果某个节点出现连接不稳定或速度慢的情况，可以尝试切换到其他节点。筋斗云拥有多个备用节点，总有一个适合你。

设置网络协议

筋斗云支持多种网络协议，包括L2TP、PPTP、OpenVPN等。不同的协议有不同的特点：

  • OpenVPN：安全性最高，但速度可能稍慢，适合对隐私要求较高的用户。
  • L2TP：速度和安全性均衡，适合大多数用户。
  • PPTP：速度最快，但安全性较低，适合对速度要求高且网络环境安全的用户。

建议用户根据实际网络情况进行选择。如果你不确定，可以先使用默认设置，或者尝试不同的协议，找到最适合你的那一个。

开启自动连接

如果你希望每次开机或连接网络时自动连接筋斗云，可以在设置中开启“自动连接”功能。这样，你就不需要每次都手动操作，非常方便。

使用分流功能

筋斗云还提供了分流功能，允许你指定哪些应用或网站通过VPN连接，哪些使用本地网络。例如，你可以让浏览器通过VPN访问海外网站，而让游戏或视频应用使用本地网络，以节省带宽和提高速度。这个功能对于需要同时使用国内外服务的用户来说非常实用。

FAQ

如何检测并处理订阅中的死节点？
定期运行批量测速脚本或使用客户端内置的 url-test、health-check 功能，自动标记或移除响应超时或重连失败的节点。结合订阅更新策略可自动替换或提醒用户维护节点列表。
Vmess 的安全加密方式推荐使用哪种？
目前推荐使用 auto 或 none（结合 TLS 加密）以提升性能。旧版 AEAD 加密已被废弃，使用 TLS/XTLS 层提供的加密即可确保安全。过度加密反而增加 CPU 占用与延迟。
如何用 systemd 管理 V2Ray/ Sing-box 服务？
创建 systemd 单元文件，指定 ExecStart 指令并设置 Restart=on-failure 与合适的 User/Group。启用服务并使用 `systemctl enable` 与 `systemctl start` 管理，便于开机自启与日志集中收集。
WinXray 的测速功能如何使用？
WinXray 内置节点测速工具，可以通过延迟测试（ping）或带宽测试判断节点质量。用户只需右键节点选择“测速”即可，测试结果将显示在列表中，帮助快速挑选最佳线路。
Mellow 是否支持自定义 DNS？
支持。Mellow 的配置文件中可设置 DNS 服务器地址及模式（如 DoH、DoT）。用户可使用安全 DNS，如 Cloudflare 1.1.1.1 或 Google DNS，以减少污染风险。
Trojan-Go 与 Trojan 有何不同？
Trojan-Go 是 Trojan 的分支版本，支持多路复用、QUIC、WebSocket 等新特性。它在兼容性和性能上都有显著提升，适合与现代客户端（如 Clash Meta、Sing-box）搭配使用。
Netch 的 TCP/UDP 分流适合哪些应用？
TCP 分流适合网页、下载和视频播放，UDP 分流适合游戏、语音和实时视频。Netch 可同时代理 TCP 和 UDP 流量，实现精准分流，保证不同应用的最佳性能。
VMess 使用 AEAD 加密的优势是什么？
AEAD 加密提供更强的安全性和抗重放攻击能力，同时隐藏流量特征，降低被 DPI 检测风险。在高封锁环境下使用，可提升连接稳定性和长期可用性。
Clash 的规则订阅更新频率如何设置？
Clash 支持设置订阅规则的更新频率，例如每小时或每天。定期更新可保证节点和规则最新，减少手动维护，提高访问速度和分流效率，同时保持网络稳定性。
VLESS + XTLS 模式在移动网络下的优势是什么？
XTLS 模式减少握手次数和加密开销，适合高延迟或不稳定的移动网络。结合策略组和多出口节点使用，可提高访问速度和连接稳定性，保证用户体验流畅可靠。
